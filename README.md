# clinica_poo
Estudo de composição em Orientação a objetos emulado a partir de código em java e ruby


### Considerações iniciais
POO é um paradigma de programação que organiza o software em "objetos". Esses objetos são como representações de coisas do mundo real, com características (atributos) e ações (métodos) que eles podem realizar.
